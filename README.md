# **Estatística com Apoio Computacional (EAC)**
Este repositório contém práticas e estudos realizados com a linguagem R na disciplina de Estatística com Apoio Computacional. O conteúdo foca no processamento de dados e aplicação de métodos estatísticos computacionais.

Técnicas de manipulação de amostras e grandes volumes de dados.

Visualização de Dados com construção de histogramas, gráficos de barras, gráficos de pizza e distribuições de frequências.

Cálculos de média, desvio padrão, variâncias e análise de quartis.

Estudo e aplicação de variáveis quantitativas e qualitativas.

Tecnologias Utilizadas:
Linguagem R
RStudio / Ambiente de desenvolvimento R
