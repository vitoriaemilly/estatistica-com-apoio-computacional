# estatistica-com-apoio-computacional
Este repositório tem como objetivo práticas e estudos da linguagem R na disciplina Estátistica com apoio computacional do S4.
-Práticas de manipulação de alto volume de dados e amostras
-Distribuições de frequências, histogramas, gráfico de barras e de pizza
-Desvio padrão, média, quartis, vâriancias
-Variáveis quantitativas & qualitativas
